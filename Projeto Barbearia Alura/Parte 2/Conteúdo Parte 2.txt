AULA 02: Navegação entre páginas

- Criar links para outras páginas, sejam elas do nosso projeto ou páginas externas, é atráves
da tag <a href=''>;

- Para transformar o texto para ter todas as letras maiúsculas é atráves do CSS com a proprie
dade 'text-transform: uppercase;'

- Para deixar o texto em negrito com CSS, é atráves da propriedade 'font-weight: bold';

- Para remover a decoração do texto, é atráves da propriedade 'text-decoration: none';

AULA 03: Posicionamento dos elementos 

- Para remover os estilos que o navegador cria automaticamente foi criado um CSS chamado re
set.css zerando todos os estilos;

- O funcionamento dos posicionamentos static, relative e absolute dos elementos:
	- Static: A imagem nao pode ser mudada de posição;
	- Relative: determina que o elemento será posicionado de acordo com o fluxo natural da
	  página.
	- Absolute: indica que o elemento ficará em uma posição fixa em relação à página.

AULA 04: A tag section

- A tag main, é usada para receber o conteúdo principal da nossa página;
- A tag section, é usada para juntar um conjunto de código de um seção da página;
- Para utilizar o inline-block é Um bloco inline é colocado inline, mas comporta-se como se
 fosse um bloco;

AULA 05: Lidando com Bordas

- Através do CSS para aplicar bordas nos elementos usa-se a propriedade 'border', etc para 
  algo específico;
- Para deixar a borda arredondada utiliza-se a propriedade border-radius;

AULA 06: Pseudo-classe CSS

- Pseudo-classes CSS:
	- hover, passar o cursor sobre o elemento ocorre uma ação;
	- active, quando um elemento está sendo ativado pelo usuário, ou seja, sendo clicado;
- Para mudar a cor do texto e/ou da borda de um elemento, quando o usuário passar o cursor 
  sobre o mesmo: 

	<h1 class='título-principal'>Olá Mundo!</h1>	

	.titulo-principal:hover{
		color: red; 
		border: solid
		border-color: red;
	}
	
- A mudar a cor da borda de um elemento, quando o mesmo estiver sendo ativado pelo usuário

	<h1 class='título-principal'>Olá Mundo!</h1>	

	.titulo-principal:active{
		color: orange; 
		border: solid
		border-color: orange;
	}
